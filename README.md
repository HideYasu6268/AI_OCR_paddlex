# AI_OCR_paddlex
ローカルLLMを使って、領収書の画像データを読み取って、CSVにする

領収書AIOCR処理ツール 取扱説明書

【概要】
このソフトウェアは、領収書の画像から日付・会社名・税率・合計金額・登録番号を自動抽出し、
CSVファイルに保存するためのツールです。

【主な機能】
1. 複数画像の一括AIOCR処理
2. 各項目の確認と手動修正
3. CSVファイルへの保存（Shift-JIS形式）

【対応画像形式】
PNG, JPG, JPEG, BMP, TIF, TIFF

【使い方】

■ ステップ1：画像の読み込み
1. 「画像を開く（複数可）」ボタンをクリック
2. 処理したい領収書画像を複数選択
3. AIによる自動解析が開始されます（全画像一括処理）

■ ステップ2：内容の確認と修正
1. 「前へ」「次へ」ボタンで画像を切り替え
2. 各項目を確認し、必要に応じて修正：
   - 日付 (date)：日付形式で入力
   - 会社/店舗 (company)：会社名や店舗名
   - 税率 (tax_rate)：税率（例: 10%）
   - 合計 (total_amount)：金額（¥記号は自動除去）
   - 登録番号 (T13桁)：T+13桁の登録番号
   - インボイス登録：登録番号が正しければ自動チェック

3. 各項目はドロップダウンから選択または直接入力可能
4. 修正後は自動的に「確認済み」状態になります

■ ステップ3：CSV保存
1. すべての画像の確認が終わったら「全て保存」ボタンをクリック
2. 保存先のCSVファイルを選択（初回のみ）
3. データがShift-JIS形式で保存されます

【出力CSVの形式】
日付,会社名,税率,合計金額,登録番号,インボイス登録
例：
2024/01/15,株式会社○○商店,10%,5500,T1234567890123,TRUE

【注意事項】

● 画像の品質について
- 文字が鮮明な画像を使用してください
- 傾きや影があると認識精度が低下することがあります

● 登録番号について
- 「T+13桁数字」の形式のみ有効です
- 正しい登録番号が入力されると「インボイス登録」が自動でONになります

● 金額について
- CSV保存時は数字のみに変換されます（¥やカンマは除去）
- 全角数字は半角に自動変換されます

● 処理中について
- 初回起動時はAIモデルの読み込みに数分かかることがあります
- 大量の画像を処理する場合は、処理完了までお待ちください

● データの保存について
- 各画像を確認後、「全て保存」でCSVに出力してください
- 同じ保存先を指定すると上書きされます



□ CSVが文字化けする
- 本ツールはShift-JIS形式で出力します
- Excelで開く場合は文字コードをShift-JISに設定してください
---
※ 本ツールはAI技術を使用していますが、100%正確な認識を保証するものではありません。
重要なデータは必ず目視で確認してください。
